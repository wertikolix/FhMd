# Orca 0.2.1

Release focus: docs completeness and code block rendering polish.

## Highlights

- Expanded `README.md` into practical documentation:
  - setup and quick start
  - parser API and options
  - supported syntax matrix
  - renderer behavior and security defaults
  - styling guide and verification commands
- Improved code block rendering in Compose:
  - rounded container with subtle border
  - optional language label when parser provides language
  - horizontal scroll for long lines (`softWrap = false`)
- Extended tests:
  - code language label normalization
  - style compatibility accessors for new code block style fields
