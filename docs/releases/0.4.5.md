# Orca 0.4.5

Release focus: code block copy button.

## Highlights

- Added optional copy-to-clipboard button in code blocks.
  - `showCopyButton: Boolean = true` in `OrcaCodeBlockStyle`.
  - Button appears in the code block header row, right-aligned next to the language label.
  - Shows "Copy" / "Copied" feedback (2s timeout).
  - Styled consistently with the language label (same background, padding, shape).
  - No external icon dependencies.
- No parser changes.
- No breaking changes.

## Usage

Enabled by default. To disable:

```kotlin
OrcaStyle(
    code = OrcaCodeBlockStyle(showCopyButton = false),
)
```
