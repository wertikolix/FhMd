# Orca 0.1.0-alpha09 (draft)

Release focus: full project rebrand from `fhmd` naming to `orca`.

## Highlights

- Renamed Kotlin API surface:
  - `FhMd*` -> `Orca*`
  - `CommonmarkFhMdParser` -> `CommonmarkOrcaParser`
  - `FhMd()` composable -> `Orca()`
- Renamed package namespaces:
  - `ru.wertik.fhmd.*` -> `ru.wertik.orca.*`
- Renamed Gradle modules:
  - `:fhmd-core` -> `:orca-core`
  - `:fhmd-compose-android` -> `:orca-compose-android`
- Compose publication coordinate is now single and short:
  - `ru.wertik:orca-compose`
- Updated CI/release workflows, sample app ids/themes/strings, and all docs to Orca naming.

## Maven Coordinates

- `ru.wertik:orca-core:<version>`
- `ru.wertik:orca-compose:<version>`

## Verification

```bash
./gradlew --no-daemon --build-cache :orca-core:test :orca-compose-android:testDebugUnitTest :sample-app:assembleDebug :sample-app:assembleRelease
```
