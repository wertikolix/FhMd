# 0.8.0

## New features

### Link title support
- `OrcaInline.Link` now includes an optional `title` field
- Titles from `[text](url "title")` and reference links with titles are preserved in the AST

### Emoji shortcode conversion
- `:smile:`, `:rocket:`, `:fire:`, and 150+ other shortcodes are automatically converted to unicode emoji
- Only applied to text nodes -- code blocks and inline code are not affected

### Superscript and subscript
- `^text^` renders as superscript (e.g. E = mc^2^)
- `~text~` renders as subscript (e.g. H~2~O)
- `~~text~~` (double tilde) remains strikethrough as before
- New AST nodes: `OrcaInline.Superscript`, `OrcaInline.Subscript`
- Configurable styles via `OrcaInlineStyle.superscript` and `OrcaInlineStyle.subscript`

### Admonition / callout blocks
- GitHub-style admonitions: `> [!NOTE]`, `> [!TIP]`, `> [!IMPORTANT]`, `> [!WARNING]`, `> [!CAUTION]`
- Parsed as `OrcaBlock.Admonition` with type, optional custom title, and content blocks
- Rendered with colored left stripe, icon, title, and nested content
- Full style customization via `OrcaAdmonitionStyle`
- Light and dark theme presets included

### Improved HTML rendering
- HTML blocks now render with styled AnnotatedString instead of plain text fallback
- Supported tags: `<b>`, `<strong>`, `<i>`, `<em>`, `<s>`, `<del>`, `<u>`, `<code>`, `<a>`, `<sup>`, `<sub>`, `<mark>`, `<br>`, `<p>`, `<h1>`-`<h6>`, `<li>`, `<hr>`, `<blockquote>`, `<pre>`
- HTML entities properly decoded
- Unknown tags gracefully stripped

### Inline image rendering
- Inline images (images mixed with text in paragraphs) now render as actual images using Compose InlineTextContent
- Uses Coil 3 AsyncImage for loading
- Configurable size and shape via `OrcaInlineImageStyle`
- Security policy applied to inline image sources

## Style changes

- Added `OrcaAdmonitionStyle` for callout block customization
- Added `OrcaInlineImageStyle` for inline image size/shape
- Added `superscript` and `subscript` span styles to `OrcaInlineStyle`
- Added `admonition` and `inlineImage` fields to `OrcaStyle`
- Updated `OrcaDefaults.darkStyle()` with dark theme admonition colors

## Sample app

- Updated to showcase all new features: admonitions, superscript/subscript, emoji shortcodes, HTML rendering
