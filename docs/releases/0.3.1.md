# Orca 0.3.1

Release focus: code block readability and usability.

## Highlights

- Improved code block rendering in `orca-compose-android`:
  - selectable code text
  - optional line numbers for multiline blocks
  - dedicated `OrcaCodeBlockStyle` controls for line numbers.
- Added tests for code line number formatting and style compatibility accessors.
- No parser behavior changes in this release.

## Maven Coordinates

- `ru.wertik:orca-core:0.3.1`
- `ru.wertik:orca-compose:0.3.1`
