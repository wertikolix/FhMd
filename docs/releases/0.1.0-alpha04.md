# FhMd 0.1.0-alpha04 (draft)

Release focus: markdown table support using pure Compose rendering.

## Highlights

- Added GFM pipe table parsing via `commonmark-ext-gfm-tables` (`0.27.1` line).
- Added table block model in `fhmd-core` AST:
  - table header
  - table rows
  - cell alignment (`left/center/right`)
- Added table rendering in `fhmd-compose-android`:
  - pure Compose implementation (`Row`/`Column` + horizontal scroll)
  - Material-friendly defaults in `FhMdStyle`
- Added parser and renderer unit tests for table parsing/mapping/alignment and inline content in cells.
- Updated sample markdown screen to show a table demo.

## Supported

- Parser backend: `commonmark-java` (`0.27.1`)
- Extensions: GFM pipe tables (`commonmark-ext-gfm-tables`)
- Public parser API:
  - `interface FhMdParser { fun parse(input: String): FhMdDocument }`
- Blocks:
  - heading
  - paragraph
  - bullet list / ordered list
  - quote
  - fenced code block / indented code block
  - table
- Inlines:
  - text
  - bold
  - italic
  - inline code
  - link
  - soft/hard line breaks
- Compose renderer:
  - `FhMd(markdown = ...)`
  - `FhMd(document = ...)`
  - `onLinkClick` callback
  - baseline styling via `FhMdStyle`

## Not Supported Yet

- images
- HTML blocks/inline HTML
- strikethrough
- task lists
- thematic breaks (`---`)
- footnotes/front matter/extensions
- non-Android targets (multiplatform-ready architecture only)
