# Orca 0.4.4

Release focus: dark theme support.

## Highlights

- Added `OrcaDefaults` with `lightStyle()` and `darkStyle()` factory methods.
  - `darkStyle()` provides a complete dark color scheme: dark code block backgrounds, adjusted syntax highlighting (VS Code-inspired), dark table headers/borders, muted quote stripes, and light-on-dark inline code.
  - `lightStyle()` returns the existing default style unchanged.
- No parser changes.
- No breaking changes.

## Usage

```kotlin
val style = if (isSystemInDarkTheme()) OrcaDefaults.darkStyle() else OrcaDefaults.lightStyle()

Orca(
    markdown = text,
    style = style,
)
```

Both presets can be customized further via `copy()`:

```kotlin
OrcaDefaults.darkStyle().copy(
    code = OrcaDefaults.darkStyle().code.copy(showLineNumbers = false),
)
```
