# FhMd

Лёгкий рендерер Markdown для Jetpack Compose.

## Что это

**FhMd** — это библиотека, которая помогает отображать Markdown-контент в Compose UI.
Подходит для экранов с заметками, статьями, справкой и in-app документацией.

## Возможности

- Отрисовка базовых Markdown-элементов в Compose.
- Простая интеграция в существующий `@Composable` UI.
- Расширяемая структура для кастомного оформления.

## Быстрый старт

1. Подключите модуль/библиотеку в проект.
2. Передайте Markdown-строку в composable-компонент рендерера.
3. Настройте тему и стили под ваш дизайн.

> Пример API может отличаться в зависимости от версии.

## Пример использования

```kotlin
@Composable
fun ArticleScreen(markdown: String) {
    // Замените на реальный composable из библиотеки
    MarkdownRenderer(
        content = markdown,
        modifier = Modifier.padding(16.dp)
    )
}
```
